Запуск в консоле Linux с пониженным приоритетом: nice -n 19
Параметры запуска скрипта:
1. логин почтового аккаунта;
2. пароль почтового аккаунта;
3. указание какую задачу будет выполнять Roundcube;
4. указание записывать лог-файл;
4. указание записывать отладочную информацию;
5. указание какие будем задействовать url-адреса для выполнения задачи;
6. амперсанд для указания ОС выполнять скрипт в фоновом режиме.
